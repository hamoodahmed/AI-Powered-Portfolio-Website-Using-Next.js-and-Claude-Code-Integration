## Overview
Built for a retail chain managing 50+ SKUs across multiple locations. The system uses machine learning to predict inventory needs, reducing both stockouts and overstock situations.

## Key Features
- **Predictive Analytics**: Uses historical sales data, seasonality, and external factors (weather, holidays) to forecast demand.
- **Real-Time Alerts**: Sends notifications to procurement teams when stock levels approach critical thresholds.
- **Cost Optimization**: Reduced inventory holding costs by 25% while maintaining 99% product availability.

## Technical Stack
- **ML Framework**: Scikit-learn, Prophet (Facebook)
- **Backend**: Python (FastAPI), Redis for caching
- **Frontend**: Next.js dashboard with real-time charts
- **Database**: PostgreSQL with TimescaleDB extension

## Business Impact
- 25% reduction in inventory costs
- 99% product availability maintained
- 2-week advance prediction accuracy: 87%
