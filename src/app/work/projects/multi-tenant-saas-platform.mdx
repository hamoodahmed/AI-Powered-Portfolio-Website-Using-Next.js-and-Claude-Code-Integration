## Overview
Built a multi-tenant SaaS application that allows small businesses to manage their operations (CRM, invoicing, inventory) from a single platform. Each tenant gets isolated data and custom branding.

## Key Features
- **True Multi-Tenancy**: Row-level security in PostgreSQL ensures complete data isolation.
- **White-Label Branding**: Each business can customize colors, logos, and domain names.
- **Role-Based Access Control**: Granular permissions for team members.
- **API-First Architecture**: RESTful APIs allow third-party integrations.

## Technical Stack
- **Frontend**: Next.js 15, Tailwind CSS, Shadcn UI
- **Backend**: Node.js, tRPC for type-safe APIs
- **Database**: PostgreSQL with Row-Level Security (RLS)
- **Auth**: NextAuth.js with JWT
- **Deployment**: Vercel (Frontend), Railway (Backend)

## Business Impact
- Serving 100+ active businesses
- 99.9% uptime over 6 months
- Average page load time: <1 second
