## Overview
As part of my specialization at PIAIC, I built a support system where multiple AI agents collaborate. One agent triages the ticket, another searches the knowledge base, and a third drafts the response.

## Key Features
- **Autonomous Reasoning**: Agents can "think" and decide when they need more information from the user.
- **Tool Use**: Integration with Zendesk and Slack APIs to perform actions directly.
- **Self-Correction**: A supervisor agent reviews all drafts before they reach a human moderator.

## Technical Stack
- **Orchestration**: LangGraph, n8n
- **LLM**: Claude 3.5 Sonnet
- **Integration**: Python, Node.js
