## Overview
Developed for a finance department processing 500+ invoices monthly. The system uses Claude Vision to read invoices in any format (PDF, images, scanned documents) and automatically enters data into the ERP.

## Key Features
- **Multi-Format Support**: Handles PDFs, JPEGs, PNGs, and even handwritten invoices.
- **Smart Validation**: Cross-checks invoice data against purchase orders and flags discrepancies.
- **Auto-Reconciliation**: Matches payments with invoices and updates accounting records.

## Technical Stack
- **AI**: Claude 3.5 Sonnet (Vision API)
- **OCR**: Tesseract + Custom preprocessing
- **Backend**: Python, Celery for async processing
- **Integration**: REST APIs to connect with QuickBooks and ERPNext

## Business Impact
- 90% reduction in manual data entry time
- 99.5% accuracy in data extraction
- Processing time reduced from 2 hours to 5 minutes per batch
