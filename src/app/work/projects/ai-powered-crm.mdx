## Overview
This project involved building a next-generation CRM for a mid-sized enterprise. The goal was to move beyond simple data entry and create a system that actively helps sales teams by providing AI-driven insights.

## Key Features
- **Automated Lead Scoring**: Uses Claude AI to analyze customer profiles and past interactions to predict conversion probability.
- **Smart Interaction Summaries**: Automatically generates concise summaries of emails and meeting transcripts.
- **Sentiment Analysis**: Tracks customer mood over time to flag potential churn risks early.

## Technical Stack
- **Frontend**: Next.js 15, Tailwind CSS, Once UI
- **Backend**: Python (FastAPI), PostgreSQL
- **AI**: Claude 3.5 Sonnet, LangChain
